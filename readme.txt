== Description ==

Extends WooCommerce by Adding the [TRANSAX](http://www.transaxgateway.com/ "TRANSAX Gateway") Gateway.
 
== Installation ==
 
1. Upload TRANSAX For WooCommerce through the WordPress Plugin module.
2. Activate TRANSAX For WooCommerce 
3. TRANSAX For WooCommerce settings can be found in the WooCommerce Settings page under the Checkout Tab
